Button Sizing Guidelines (Web & Mobile)

We define separate button sizes for web and mobile, differing only in height and minimum width to meet touch target standards. These follow best practices and Apple Human Interface Guidelines for accessibility.

  • Web: Min width 44px, min height 44px
  • Mobile: Min width 48px, min height 48px

This ensures a consistent and user-friendly experience across devices.
